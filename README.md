# StudyHub v2 Capstone

> SoftUni "Full Stack Apps with AI" course capstone project.

## Monorepo Structure

```text
apps/
  web/      Next.js web app + API routes
  mobile/   Expo React Native app
packages/
  shared/   Shared TypeScript types/utilities
docs/       Assignment + implementation docs
```

## Quick Start

1. Install dependencies: `npm install`
2. Copy env template: `copy .env.example .env`
3. Run web app: `npm run dev:web`
4. Run mobile app: `npm run dev:mobile`

## Current Status

Phase 0 bootstrap is in place (workspace setup, app skeletons, shared package).
